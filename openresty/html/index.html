pgAdmin

To access the database started by Docker Compose you need to know the IP address.

First you need to have the container ID that you can get by running this command:

docker ps

Next run this command to get the IP address:

docker inspect <container id> | grep IPAddress

Use this IP address for the Host name/address in pgAdmin.
